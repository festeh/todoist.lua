# todoist.lua

An unofficial Todoist client for Neovim.

Achtung! Work in progress.

## Installation

### Lazy.nvim
```
  {
    'festeh/todoist.lua',
    dependencies = {
      'nvim-lua/plenary.nvim',
      'muniftanjim/nui.nvim',
    }
  },
```

## Usage
Set `TODOIST_API_KEY` environment variable. Then open Neovim and run `:Todoist`
